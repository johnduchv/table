Actors Administrator
Description Ca sử dụng miêu tả cách một admin thêm phim vào database
Trigger Khi admin muốn thêm phim mới vào database
Pre-conditions 1. Cần biết thông tin cổng và tài khoản của admin 2. Phim chưa được tồn tại trong database
Post-conditions Phim được thêm thành công vào database
Normal Flow 1. Admin vào cổng localhost và đăng nhập theo tài khoản của admin 2. Admin chọn Movies và nhấn vào Create 3. Admin điền đầy đủ thông tin cần cho 1 phim 4. Admin ấn Create để xác nhận tạo phim
Alternative Flows N/A
Exception • Phim không tồn tại trong database Admin bị mất kết nối mạng
