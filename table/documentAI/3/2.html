Actors Administrator
Description Ca sử dụng miêu tả cách một admin xoá đi một phim trong database
Trigger Khi admin cần xóa một phim
Pre-conditions • Cần biết thông tin cổng và tài khoản của admin Phim đã tồn tại trong database •
Post-conditions Phim được xoá thành công
Normal Flow 1. Admin vào cổng localhost và đăng nhập theo tài khoản của admin 2. Admin chọn Movies, tìm phim muốn xoá 3. Admin tìm đến phim muốn xoá 4. Admin ấn vào icon thùng rác
Alternative Flows N/A
Exception • Phim không tồn tại trong database Admin bị mất kết nối mạng
